Implementar una página HTML estática en Vercel


Vercel es un sitio de servicios en la nube.

Proporciona alojamiento para apps creadas con tecnologías como:

  • Node.js
  • Ruby
  • Python
  • Go

Requisitos para implementar una aplicación en Vercel

  • Una cuenta de Vercel
  • Git instalado en computador
  • CLI de Vercel instalada en computador

Pasos

1) Tener una carpeta con un proyecto hlml con: index.html, style.css, images, etc.

2) https://nodejs.org/es/download/ instalar node

En la terminal o bash:

3) posicionarse en la carpeta

4) git init iniciar git

5) git add . agregar cambios

6) git commit -m "desplegando" comitear cambios

7) npm i -g vercel instalar vercel

8) vercel iniciar vercel en el directorio

9) seguir las instrucciones

10) Aquí está el enlace de tu app: https://nombre_de_app.vercel.app

11) vercel --prod actualizar cambios a Vercel
